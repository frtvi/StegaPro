# StegaPro
Esteganografia automatizada<br>Este é um programa simples em Python que permite esconder um arquivo em uma imagem utilizando a técnica de esteganografia.

## Requisitos

- Python 3.x
- Biblioteca Pillow (`pip install Pillow`)
